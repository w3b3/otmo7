Surface your contribution.

It is not by accident or chance that we are all here. It must be intentional.

We all win if we all share the same boat — rowing on the same cadence, towards the same end goal.

  • If a day of effort produced no artifact — something went wrong. Either the work wasn't worth it, or the output wasn't captured.
  • If a PR merged but no signal moved — the work isn't done.
  • Both are problems worth surfacing.
